# HeroAPI
Vert.X (Java) PoC (Proof of concept)

# Build/Run

### Compile (JDK 1.8.x)
    mvn clean compile 

### Build (jar)
    mvn clean package
    
### Run
We manage it in 2 different ways:
#### through Docker (you need to have it installed locally)
    ./docker run
